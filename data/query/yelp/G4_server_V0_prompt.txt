# Server Performance Assessment

Help assess the front-of-house staff performance at this restaurant using customer reviews as evidence.
This is from the perspective of a business owner evaluating their service team.
Assign a performance label: Needs Improvement, Satisfactory, or Excellent.

## Definitions of Terms

### Incident

Consider reviews that mention any of the following:
- server attentiveness and responsiveness
- friendliness and demeanor
- menu knowledge and recommendations
- service errors and how they were handled
- exceptional servers mentioned by name

Reviews that only discuss food without mentioning service carry less weight.

### Account Type
How the reviewer relates to the described event

- **Firsthand**: The reviewer or their dining party experienced the event directly (e.g., "I had...", "my child...")
- **Secondhand**: The reviewer reports someone else's experience (e.g., "my friend...", "I heard...")
- **Hypothetical**: Concern or preference without describing an actual incident (e.g., "I'm allergic so I worry...")

### Attentiveness
How attentive was the server

- **Absent**: Server was nowhere to be found
- **Poor**: Server rarely checked on table
- **Adequate**: Acceptable level of attention
- **Good**: Checked in appropriately, responsive
- **Excellent**: Anticipatory service, exceptional attention

### Friendliness
Demeanor of the server

- **Rude**: Actively rude or hostile
- **Cold**: Distant, unfriendly manner
- **Neutral**: Neither friendly nor unfriendly
- **Friendly**: Warm and personable
- **Warm**: Exceptionally warm, made guests feel special

### Menu Knowledge
Server's knowledge of menu and ingredients

- **None**: No knowledge, couldn't answer questions
- **Poor**: Limited knowledge, many uncertainties
- **Adequate**: Basic knowledge, can handle common questions
- **Good**: Good knowledge with recommendations
- **Excellent**: Expert knowledge, excellent recommendations

### Service Errors
Errors made by the server

- **None**: No service errors
- **Minor**: Small mistakes quickly corrected
- **Significant**: Notable errors affecting experience
- **Severe**: Major errors significantly impacting experience

**Needs Improvement** if Excellent does not apply, but any of the following is true:
- {'text': 'Multiple reviews describe inattentive or absent servers', 'type': 'count_threshold', 'filter': {'attentiveness': 'low'}, 'min_count': 2}
- {'text': 'Consistent complaints about rude or unfriendly service', 'type': 'count_threshold', 'filter': {'friendliness': 'rude'}, 'min_count': 2}
- {'text': 'Frequent service errors reported', 'type': 'count_threshold', 'filter': {'service_errors': 'frequent'}, 'min_count': 2}

**Satisfactory** otherwise, especially when:
- Service is acceptable but not exceptional
- Mixed reviews about server quality
- Some good experiences, some issues
